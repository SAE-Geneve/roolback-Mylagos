# RollbackGame
Documentation: https://eliasfarhan.github.io/rollback_doc/ 

Kill your enemy by crushing them with blocks

Controls p1 :
w = jump,
a = left,
d = right

q = move spawner left,
e = move spawner right,
s = drop block

Controls p2 : 
i = jump,
j = left,
l = right

u = move spawner left,
o = move spawner right,
k = drop block
